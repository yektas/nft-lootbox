## About The Project

Completed code of NFT Lootbox with Thirdweb tutorial on [@pointer.gg](https://www.pointer.gg/tutorials/thirdweb-nft-lootbox)

## Built With

- NextJS
- Thirdweb
- TailwindCSS

## Installation

1. Clone the repo
   ```sh
   git clone https://github.com/yektas/nft-lootbox.git
   ```
2. Install packages

   ```sh
   yarn install
   ```

3. Run the application

   ```sh
   yarn dev
   ```
